# Vincent OS Scripts
This collections of scripts are used to automate multiples functions of differents Vincent OS Editions.

- PowerShell scripts are for Vincent OS Standard Edition.
- Bash scripts are for Vincent OS Legacy Edition.

## Installation
Collections of scripts are already installed in Vincent OS Editions.

Name of the package in the Vincent OS repositories (all Editions): ``scripts``

## Contribute
Vincent OS Scripts is open to all contributions! You can read more at the CONTRIBUTE.md file.
